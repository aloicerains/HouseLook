HouseLook API

Let's find you a new House!

The ultimate goal of the website it help users find new houses in different towns and cities. The first part is is a house searching API developed with fake data for learning purposes. The API provides resource routes description to use methods such as:

  • GET
  • PUT
  • POST
  • DELETE

Resources

The API currently have 6 major resources:

  • /places
  • /houses
  • /rooms
  • /owners
  • /users
  • /reviews

Guide to access resources

Places
